Oversimple
Oversimple is a library for audio OVERsampling with a SIMPLE api
oversimple

Oversimple is a C++17 library for audio oversampling, which tries to offer a simple api.

Oversimple wraps two of the best resampling libraries available:

  • HIIR, by Laurent De Soras for minimum phase antialiasing, through my fork which adds support for double precision floating-point numbers, and AVX instructions. HIIR only implements power of two resampling.
  • and r8brain-free-src by Aleksey Vaneev, for linear phase antialiasing, through my fork, which adds supports for my fork of Julien Pommier's PFFFT library, which can work with double-precision floating point numbers using AVX instructions.

Aligned memory and interleaved buffers needed by the simd code in HIIR are managed using avec.

Usage

Just add everything to your project except for the content of the test folders and the .cpp files in audio-vec/vectorclass.

If you support AVX instructions, you may want to define R8B_PFFFT_DOUBLE=1 in r8bconf.h or as a preprocessor definition. See r8brain/README.md for more details.

Dependencies

  • Boost.Align.
  • pthread on *nix (only r8brain).
