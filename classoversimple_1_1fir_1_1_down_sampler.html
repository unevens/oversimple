Detailed Description

DownSampler using a FIR antialiasing filter.

Its processing method takes a number of requested samples, and will output either output that much samples, or no samples at all. It uses a buffer to store the samples produced.

See also
TUnbufferedReSampler for a template that can work with single precision.
